# Post Build Checklist

## Windows
- instala sem erro
- abre sem crash
- backend arranca
- `/docs` responde
- login funciona
- projeto abre
- outputs aparecem

## Android
- APK instala
- app abre
- UI carrega
- ligação funciona
- updater abre link correto

## Funcional
- story layout abre
- ilustração abre
- preview de voz toca
- audiobook exporta
- vídeo exporta
- updater encontra versão remota

## Final
- release notes atualizadas
- version.json atualizada
- JSON remoto do updater publicado
