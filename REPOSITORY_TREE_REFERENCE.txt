==================================================
ARCHITECTURAL READING OF THIS TREE
==================================================
- Studio is a private monorepo.
- Frontend/mobile/desktop shell lives under client/mobile-app.
- Backend/API core lives under studio_core.
- Public/static brand media lives under public.
- Automation/local tooling lives under scripts.
- Publication contract, public assets, storefront, branding, visuals, and pairing are already separated as independent route/service concerns.

==================================================
TREE GOVERNANCE RULE
==================================================
Before structural refactors:
1. Keep this repository modular.
2. Add new integration services alongside existing modules.
3. Redirect legacy code paths before deleting them.
4. Preserve green builds.
5. Do not merge this repository with Website.
