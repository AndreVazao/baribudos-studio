# External Channels Roadmap

## Objetivo
Preparar a próxima camada real de distribuição externa sem partir o fluxo principal Studio → Website.

## Regra base
- O Studio continua a ser o centro de comando.
- O Website continua a ser o primeiro canal próprio de monetização.
- Amazon, YouTube e outros canais entram como expansão controlada.

## Canais-alvo
### Amazon KDP
Uso principal:
- ebooks
- expansão editorial

O Studio deve preparar:
- metadados
- capa
- copy comercial
- estado de readiness
- decisão de publicação

### YouTube / YouTube Kids
Uso principal:
- trailers
- séries
- descoberta audiovisual

O Studio deve preparar:
- vídeo principal
- título
- descrição
- readiness de publicação
- decisão editorial

### Audiobook / outras plataformas
Uso principal:
- expansão áudio
- descoberta complementar

O Studio deve preparar:
- assets áudio
- copy
- readiness por canal

## Ordem correta
1. Website próprio
2. primeiras vendas reais
3. Amazon
4. YouTube
5. outros destinos

## Meta da v3
Não é fechar integração automática total já.
É deixar o sistema explicitamente pronto para essa expansão sem confundir o papel do Website e do Studio.
