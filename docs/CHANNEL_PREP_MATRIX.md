# Channel Preparation Matrix

## Objetivo
Ver claramente o papel de cada canal e o nível de preparação necessário antes de distribuir.

## Website próprio
- prioridade: imediata
- função: descoberta, pré-lançamento, venda direta
- controlo: Studio publica para Website

## Amazon KDP
- prioridade: seguinte
- função: expansão de ebooks
- controlo: Studio prepara metadados, ficheiros e decisão de publicação

## YouTube / YouTube Kids
- prioridade: seguinte
- função: trailers, séries, descoberta audiovisual
- controlo: Studio prepara vídeo, copy e decisão de publicação

## Audiobook / outras plataformas
- prioridade: seguinte
- função: expansão editorial e áudio
- controlo: Studio prepara ativos, metadados e estado de distribuição

## Regra
Começar pelo Website próprio. Só depois escalar para os outros canais.
