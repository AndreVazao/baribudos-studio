# Sales Readiness Plan

## Objetivo
Deixar o sistema pronto para que, assim que a produção estiver boa, o fluxo de publicação e venda possa arrancar com o menor atrito possível.

## Princípio
- O Studio controla produção, estado editorial e distribuição.
- O Website é o canal próprio de descoberta e venda.
- Destinos externos entram depois, mas continuam a ser comandados pelo Studio.

## Prioridades desta fase
1. Distribution Hub mais concreto no Studio
2. readiness comercial por projeto
3. checklist de lançamento comercial
4. preparação para destinos externos

## O que um projeto deve ter para vender
- headline forte
- subtítulo claro
- cover definido
- galeria mínima
- CTA coerente
- estado público coerente
- produto ativo ou caminho de compra definido

## Estados práticos
- draft
- internal_review
- teaser_ready
- prelaunch_public
- launch_ready
- published

## Meta desta fase
Não é automatizar Amazon e YouTube por completo já. É deixar o Studio pronto para decidir e preparar distribuição logo que a produção fique pronta.
