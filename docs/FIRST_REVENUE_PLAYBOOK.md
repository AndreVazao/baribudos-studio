# First Revenue Playbook

## Objetivo
Transformar uma produção boa em primeiras vendas com o menor atrito possível.

## Sequência curta
1. fechar qualidade mínima do projeto
2. validar sales readiness no Studio
3. publicar para Website
4. validar homepage, descoberta e loja
5. acompanhar primeiras vendas
6. só depois escalar

## O que não fazer nesta fase
- abrir demasiados canais ao mesmo tempo
- publicar sem cover, headline e CTA fortes
- usar destinos externos antes de validar o Website próprio

## Regra de ouro
Primeiro provar receita no Website próprio. Depois usar Amazon, YouTube e outros canais como multiplicadores.
