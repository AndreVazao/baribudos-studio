# Project Launch Checklist

## Objetivo
Ter uma checklist curta para decidir se um projeto pode avançar para venda e distribuição inicial.

## Checklist mínima
- headline forte definida
- subtítulo claro definido
- CTA definido
- cover principal definida
- galeria mínima pronta
- estado público coerente
- projeto marcado como pronto para publicação
- sync Website validado quando já houve publish

## Interpretação
### Pode avançar
Quando a maior parte destes pontos estiver fechada e o resultado visual for forte.

### Ainda não deve avançar
Quando faltar base comercial ou visual e a publicação puder queimar o primeiro impacto.

## Regra prática
Primeiro vender bem no Website. Depois expandir para outros canais.
