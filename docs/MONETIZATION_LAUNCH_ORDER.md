# Monetization Launch Order

## Objetivo
Gerar as primeiras vendas o mais cedo possível sem abrir demasiadas frentes ao mesmo tempo.

## Ordem recomendada
1. Studio pronto para produção e decisão editorial
2. Website próprio pronto para descoberta e compra
3. primeiro projeto publicado no Website
4. validação de primeiras vendas
5. só depois expansão para Amazon, YouTube e outros canais

## Porque esta ordem é a certa
- menos dependências externas
- menos bloqueios de plataforma
- mais velocidade até à primeira receita
- mais controlo do funil completo

## Regra prática
O Website próprio é o primeiro motor de monetização. Os canais externos entram como multiplicadores, não como ponto de arranque.
