# V5 Finalization Scope

## Objetivo
Empurrar o Baribudos Studio da fase de modelação para a fase de operação quase final, preparando a base para implementação real de Distribution Hub persistido e ações por destino.

## Foco da V5
- persistência real por destino
- ações por destino
- histórico de tentativas
- notas e erros por destino
- preparação de backend para operação real

## O que esta V5 deve deixar encaminhado
1. modelo de dados por destino
2. ações manuais por destino no Studio
3. estado persistido no backend
4. base para Amazon, YouTube e áudio

## Regra
O Studio continua a ser o centro de comando. O Website continua a ser o canal próprio principal.
