# Access and Roles

## Regra base
O Baribudos Studio é uma ferramenta pessoal e familiar.

## Administradores totais
Os logins com controlo total do Studio e do Website são:
- André
- esposa do André

## Poder dos administradores
Administradores totais podem:
- entrar no Studio com acesso completo
- controlar publicação para o Website
- decidir estado editorial e estado público
- promover pré-lançamento e lançamento
- controlar readiness comercial e distribuição

## Produção sem controlo de publicação
Os filhos podem usar o Studio para:
- produção
- preparação de assets
- tarefas operacionais de criação

Mas não devem controlar:
- publicação para o Website
- decisões finais de lançamento
- controlo administrativo total

## Regra prática
Produção pode ser distribuída pela família. Publicação e controlo comercial ficam só com os dois administradores principais.
